# Images

이 폴더는 예제 실행에 사용되는 이미지 파일을 저장합니다.

## Contents

| File            | Description                      |
| --------------- | -------------------------------- |
| sample.png      | Sample image for OpenCV examples |

## Notes

- 저작권 문제가 없는 이미지 또는 직접 촬영한 이미지만 사용합니다.
- 회사 프로젝트 이미지 및 고객 데이터는 포함하지 않습니다.
- 예제 실행을 위한 최소한의 이미지 파일만 저장합니다.
